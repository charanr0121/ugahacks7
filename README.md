# Both at same time

```
npm run dev
```

# Individual

To start flask server:
```
cd api
```
first time running:
```
export FLASK_APP=hello
```
```
flask run
```

then go to [localhost:5000](localhost:5000)



For react app:
```
cd client
```
```
npm run start
```

then go to [localhost:3000](localhost:3000)
